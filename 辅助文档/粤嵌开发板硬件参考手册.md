## 1.2 RK3568 核心板资源  

从图 1.2.1 可以看出， RK3568 核心板板载资源丰富，可以满足各种应用的需求。整个核心板采用了 314P 金手指接口，使得其可以很方便的应用在各种项目上。

粤嵌3568F核心板板载资源如下：

◆ CPU： 采用瑞芯微 RK3568 主控， 4 核 Cortex-A55，主频高达 2.0GHz。内置 H264/H265硬件编解码，支持 4K@60fps 解码， 1080P@60fps 编码， 支持高质量的 JPEG 编解码。内置 3D GPU(Mali-G52),支持 OpenGL ES1.1/2.0/3.2、 OpenCL2.0 和 Vulkan 1.1。另外内置 1.0TOPs NPU，支持 INT8 和 INT16。内置 8M ISP， 支持多屏异显。

◆ 外扩 1 片 LPDDR4/LPDDR4X， 内存容量为 2GB，位宽 32 位。

◆ 16GB EMMC 存储芯片。

◆ 314P金手指接口， 共引出127个GPIO(可复用为其他功能)、105个其他功能引脚(HDMI、EDP、 USB、 MIPI 屏幕， MIPI 摄像头、 ADC、 音频等)、 14 个电源引脚(给核心板供电引脚以及核心板 PMIC 输出电源引脚，不包含 GND)。

粤嵌RK3568F核心板的特点包括：

1)、 体积小巧。核心板仅 82mm*64mm 大小，方便使用到各种项目里面。

2)、 集成方便。核心板使用 314P 金手指接口，可以非常方便的集成到客户 PCB 上，更换简单，方便维修测试。

3)、 资源丰富。核心板板载： 2GB/4GB LDDR4/4X 内存、 16GB EMMC 存储器，可以满足各种应用需求。

4)、 性能稳定。核心板采用 10 层板沉金设计，单独地层、电源层，且关键信号采用等长线走线，阻抗匹配，保证运行稳定、可靠。



## 2.1 RK3568 开发板硬件资源说明  

**1、 DC12V电源输入**

这是开发板板载的一个外部电源输入口（DC_IN），采用标准的直流电源插座。开发板板载了 DC-DC 芯片，用于给开发板提供高效、稳定的电源 。

**2、 千兆网络接口 ETH0**

这是开发板板载的 1 个 1000M 以太网接口，可以进行 10/100/1000M 网络通信。 RK3568内部含有2个10/100/1000M以太网MAC外设，通过外接1000M网络PHY芯片，可以实现1000M有线网络。

**3、 HDMI输出接口**

这是HDMI 输出接口， RK3568 集成了一路 HDMI 2.0 显示接口，支持 10bit 色深， 1080P@120Hz 或 4096x2304@60Hz。可以通过此接口连接 HDMI 显示器，实现高清显示输出。

**4、 CAN 接口**

这是开发板板载的 CAN 总线接口（CAN），通过 2 个端口和外部 CAN 总线连接，即 CANH和 CANL。这里提醒大家： CAN 通信的时候，必须 CANH 接 CANH， CANL 接 CANL，否则可能通信不正常。

**5、 mSATA 硬盘接口**

这是开发板的 MSATA 固态硬盘接口。

**6、 调试串口(Type-C)**

这是开发板板载的一个 USB Type-C 接头（USB_TTL），用于 USB 连接 CH340C 芯片，从而实现 USB 转串口。同时，此 USB Type-C

接头也可以给开发板供电。从而实现 RK3568 开发板与电脑之间串口通信，可以通过这个串口调试开发板。  

**7、 USB OTG 接口**

这是开发板板载的一个 USB Type-C 接口，用于实现 OTG 功能，一般通过此接口来烧写系统。

**8、六轴传感器** 

开发板板载六轴传感器芯片（U20），型号为 MPU6050，通过 I²C 接口与 RK3568 处理器连接。 MPU6050 内部集成三轴加速度传感器与三轴陀螺仪，可实现姿态检测、运动感知等功能，广泛应用于姿态解算、惯性测量等场景，支持用户开展相关算法学习与项目开发。

**9、 USB3.0 HOST 接口**

这是开发板的 2路 USB3.0 HOST 接口， RK3568 有 2 个 USB3.0 HOST 接口，其中一路用于连接 5G 模组。这是另外一路，作为 USB3.0 HOST 接口，可以连接 USB 鼠标、键盘、 U 盘等设备。

**10、 USB2.0 HOST 接口**

这个是开发板上三个 USB2.0 HOST 接口， RK3568 原生支持 2 路 USB2.0 HOST 外设，速度会比上面的 USB3.0 慢。可以通过这两个 USB2.0 HOST 接口连一些低速 USB 设备。

**11、耳机接口**

这是开发板板载的 3.5mm 耳机接口。

**12、 MIC(咪头)**

这是开发板的板载录音输入口（MIC）， 核心板上的 RK809 这个 PMIC 芯片集成了音频编解码功能， 该咪头连接到 RK809 的 MIC 输入引脚上，可以用来实现录音功能。

**13、红外接收模块**

这是开发板的板载红外接收头，可以用来做红外遥控器实验。

**14、 LVDS 屏接口**

这是 LVDS 输出接口， RK3568 集成了一路 LVDS 接口，可以连接 LVDS 屏幕，最高支持4Lane 的 LVDS 屏幕，单 Lane 最高 1.2Gbps 速率。由于显示需要通过 VOP，对于 LVDS 接口VOP 支持 8bit、 1280x800@60Hz。

**15、 MIPI 摄像头接口**  

这是开发板的 MIPI CSI 摄像头接口，支持 4 Lanes，用于连接 MIPI 摄像头，比如正点原子出品的 ATK-MCIMX335 和 ATK-MCIMX415 摄像头。  

**16、按键**

这是开发板板载的6 个机械式输入按键， 其中 2个功能按键， PWRON 和 RESET。 PWRON 按键是核心板上 RK809 这颗 PMIC 的上电按键，当RK3568 不正常工作的时候可以按下 PWRON 按键重新上电。 RESET 是复位按键，用于复位整个开发板。

另外 4 个按键是用户按键，采用 ADC 的方式实现， 可以做为普通按键输入使用，默认用作菜单和音量加减按键

**17、用户 LED 灯**

这是开发板板载的 4个 LED 灯， 为白色，用户可以使用这个 LED 灯。在调试代码的时候，使用 LED 来指示程序状态，这是非常不错的一个辅助调试方法。

**18、核心板接口**

这是开发板底板上面的核心板接口， 为 314P 金手指接口座， 可以用来插RK3568F核心板，从而学习 RK3568 芯片的开发。

**19、SPK喇叭接口**

2针spk喇叭。

**10、 Nano SIM 卡接口**

这是开发板上的 Nano SIM 卡接口，如果要使用 5G 模块的话就需要在此接口中插入 NanoSIM 卡。

**21、 TF 卡接口**

这是开发板板载的一个标准 TF 卡接口（TF_CARD），采用小型的 TF 卡接口， SDMMC 方式驱动，有了这个 TF 卡接口，就可以满足大容量数据存储的需求。
